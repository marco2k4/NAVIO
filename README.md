## How It Works

### Navigation
- `ui.js → initNavPills()` wires the top nav to `showSection(id)`
- Each section has `id="section-{name}"` and shows/hides via `.active` class

### Schedule
- `data.js` holds `SCHEDULE` object keyed by day (`mon`, `tue`, etc.)
- `schedule.js → renderSchedule(day)` builds cards and injects them into `#classes-list`
- Clicking a card calls `modal.js → openModal(classData)`

### Modal (Flip Card)
- Front face: class details (time, room, building, instructor)
- "Navigate to Room" button → `flipCard()` rotates the card 180° via CSS `rotateY`
- Back face: QR code generated by `qrcodejs` library, encoded with class info
- Tapping QR → `showMapPopup()` opens the floor plan overlay
- "← Back" button → `unflipCard()`

### QR Code
- Generated using `qrcodejs` (loaded from CDN)
- Encoded data format: `NAVIO:{code}:{room}:{building}:{time}{ampm}`
- Example: `NAVIO:CPRG 307:Rm 204:Heritage Hall:8:00AM`

## Adding / Editing Classes

Open `js/data.js` and edit the `SCHEDULE` object.

Each class entry:
```js
{
  id: 1,
  name: 'Database Systems',   // Full course name
  code: 'CPRG 307',           // Course code
  time: '8:00',               // Start time
  ampm: 'AM',                 // 'AM' or 'PM'
  duration: '1h 30m',         // Duration display string
  room: 'Rm 204',             // Room label
  building: 'Heritage Hall',  // Building name
  instructor: 'Sarah Chen',   // Instructor full name
  type: 'Lecture',            // 'Lecture' | 'Lab' | 'Seminar'
  color: 'blue',              // 'blue' | 'teal' | 'purple' | 'amber' | 'rose'
}
```
